# Capacitor Airship Plugin

A Capacitor plugin for Airship's iOS and Android SDK.

## Requirements

| Platform | Requirements |
|----------|-------------|
| iOS | iOS 16.0+, Xcode 16+ |
| Android | compileSdkVersion 36+, minSdkVersion 23+, Kotlin 2.2.20+, Gradle 8.13+, JDK 21+ |
