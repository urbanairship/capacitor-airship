### Running this example

1. Install capacitor cli and core globally:
```bash
npm install @capacitor/cli @capacitor/core
```

2. Install dependencies from ROOT directory:
```bash
npm i && npm run build
cd example
npm i && npm run build
```

3. Run platform of your choice:

Android:
```bash
npx cap run android
```

iOS:
```bash
npx cap run ios
```
